# Tank Wars

## Testing & Verification
- This project (Tank Wars) is a browser-based HTML/JS/CSS game. After implementing any gameplay or physics change, verify it in the browser with before/after checks.
- Chrome throttles background tabs during automated testing; keep the game tab in the foreground when reproducing timing/physics bugs like the falling-tank issue.

## Git Workflow
- Never push directly to main. Always create a feature branch and open a PR for changes.
